Welcome

Welcome to the land of testing! There are a lot of bugs still, so let me know if you find anything! Read on to find beginner info, followed by bugs, followed by future enhancements. :)

Beginner Info

  • Enter a username and password above and hit enter to log in. Note: This is not over HTTPS yet, so don't use any passwords you care about! Passwords are not stored in plaintext.
  • If the account does not exist, it will be created with the username and password.
  • Once you are logged in, you will be able to post a message update at the bottom of the screen. Just type a message and then hit enter!
  • New messages are posted at the bottom of the page.
  • There are convenient "Top" and "Bot" buttons located in the top right and bottom right of the screen. These will allow you to quickly jump to the top and bottom of the messages.
  • When a new message is posted, you will automatically scroll to the bottom of the screen.
  • Type "/header Title Message" (no quotes) in order to create a new title heading.
  • Type "/act Action message." (no quotes) in order to submit a user action. These will appear in bold.
